# 1. Introduction

Welcome to **AyurScan**, an AI-powered medicinal plant identification platform.

This manual helps users understand how to use the application, upload plant images, view AI predictions, and manage their account.

- Stable internet connection recommended

# 3. User Registration

### Steps

1. Open AyurScan.
2. Click **Register**.
3. Enter:
   - Full Name
   - Email Address
   - Password
4. Click **Create Account**.
5. Login using your registered credentials.

---

# 4. User Login

### Steps

1. Open the Login page.
2. Enter your email.
3. Enter your password.
4. Click **Login**.

After successful authentication, the Dashboard will open.

---

# 5. Dashboard

The dashboard provides access to:

- Upload Plant Image
- Prediction History
- Favorite Plants
- User Profile
- Logout

---

# 6. Uploading a Plant Image

### Supported Formats

- JPG
- JPEG
- PNG

### Maximum File Size

10 MB

### Steps

1. Click **Upload Image**.
2. Select a plant image.
3. Click **Predict**.
4. Wait for the AI prediction.

# 12. Common Errors

## Invalid Login

Cause:

Incorrect email or password.

Solution:

Verify your credentials and try again.

---

## Unsupported Image

Cause:

Unsupported file format.

Solution:

Upload a JPG, JPEG, or PNG image.

---

## Large Image

Cause:

Image exceeds 10 MB.

Solution:

Compress the image before uploading.

---

## Prediction Failed

Cause:

Poor image quality or unsupported plant.

Solution:

Upload a clearer image containing one of the supported medicinal plants.

# 13. Frequently Asked Questions (FAQ)

### Q1. Which plants are supported?

- Tulsi
- Neem
- Aloe Vera
- Giloy
- Ashwagandha

---

### Q2. Can I upload multiple images?

No. Version 1.0 supports one image per prediction.

---

### Q3. Can I use the application without logging in?

No. Authentication is required to use prediction features.

---

### Q4. Is my data secure?

Yes. Passwords are securely hashed, and authenticated APIs are protected using JWT.

# 15. Best Practices

- Capture clear images in good lighting.
- Avoid blurry or low-resolution photos.
- Ensure only one medicinal plant is visible in the image.
- Use the latest version of a supported browser.
- Log out after using the application on shared devices.
