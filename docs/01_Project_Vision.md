# AyurScan

## Enterprise AI-Powered Medicinal Plant Intelligence Platform

Version: 1.0

Author: Shrasti Chaudhary

Date: July 2026

---

# 1. Vision Statement

AyurScan aims to become an enterprise-grade Artificial Intelligence platform capable of accurately identifying medicinal plants using Computer Vision while providing scientifically structured botanical knowledge through a modern, scalable software architecture.

Unlike conventional plant identification systems that attempt to classify hundreds of species with inconsistent accuracy, AyurScan intentionally focuses on five carefully selected medicinal plants to achieve highly reliable predictions using a curated dataset and transfer learning.

The platform combines Artificial Intelligence, modern software engineering, cloud-native architecture, secure authentication, containerization, and explainable AI into a single production-ready solution suitable for healthcare education, research, and medicinal plant awareness.

---

# 2. Problem Statement

Current medicinal plant identification applications often suffer from one or more of the following limitations:

- Low prediction accuracy due to large classification scope.
- Lack of explainability behind AI predictions.
- Limited information regarding medicinal usage and dosage.
- Poor software architecture unsuitable for enterprise deployment.
- Absence of secure authentication and scalable backend services.

These limitations reduce user confidence and limit practical adoption.

AyurScan addresses these challenges by prioritizing prediction accuracy, software quality, and maintainability over the number of supported plant species.

# 5. Objectives

## Primary Objectives

- Achieve greater than 98% plant classification accuracy.
- Identify five medicinal plant species with high confidence.
- Deliver prediction results within two seconds.
- Provide scientifically structured medicinal information.

## Technical Objectives

- Build a microservice-based architecture.
- Use PostgreSQL as the primary relational database.
- Implement REST APIs using NestJS.
- Develop an AI inference service using FastAPI.
- Containerize the application using Docker.
- Document APIs using Swagger.
- Follow clean architecture principles.

# 7. Scope

AyurScan Version 1.0 supports recognition of the following medicinal plants:

- Tulsi
- Neem
- Aloe Vera
- Giloy
- Ashwagandha

For every successful prediction the system provides:

- Plant Name
- Scientific Name
- Confidence Score
- Medicinal Benefits
- Recommended Dosage
- Active Medicinal Compounds
- Side Effects
- Contraindications
- Grad-CAM Visualization
